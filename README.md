# Little Burger Shop

A small multi-page burger restaurant website built with vanilla HTML, CSS, and a little JavaScript.

## Overview

This project was created to practice and showcase:

- Semantic HTML
- Responsive CSS layout
- Basic accessibility patterns
- Simple DOM updates with JavaScript

## Pages

- Home page with burger menu section
- About page
- Hours page
- Contact page

## Tech Stack

- HTML5
- CSS3
- JavaScript

## Run Locally

Open index.html in a browser.
